# AI Analyst Copilot Dashboard

An interactive data analytics dashboard built using Python and Streamlit.

## 🚀 Features
- Upload CSV or Excel files
- KPI Dashboard (Sales, Quantity, Avg Order Value, Peak Hour)
- Dynamic filters (Country, Year)
- Automated business insights:
  - Peak sales hour
  - Top country by sales
  - Weekday vs weekend analysis
- Visualizations:
  - Histogram
  - Scatter Plot
  - Correlation Heatmap
- Executive summary generation
- Download cleaned dataset

## 🛠 Tech Stack
- Python
- Pandas
- Streamlit
- Matplotlib
- Seaborn

## ▶️ Run locally
```bash
pip install -r requirements.txt
streamlit run app.py
